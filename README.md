# Grocery List

A simple grocery list that lets you add and delete grocery items and also helps you check the availability of the item in your nearest grocery store.

## Technologies Used

-   HTML5
-   SASS
-   JAVASCRIPT
-   MONGODB
-   EXPRESS
-   REACT
-   NODE

## Website used to Deploy

-   Backend - Heroku
-   Frontend - Netlify
